== Description ==

Enables free input for fixed rate shipping options, like:

*   pickup - $0
*   regular - $5
*   overnght - $10
*   etc..

== Installation ==


1. Upload files to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to Store->Shipping, Fixed rate will appear in Shipping modules list.
